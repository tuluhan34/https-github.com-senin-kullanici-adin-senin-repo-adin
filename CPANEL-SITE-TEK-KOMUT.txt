ZIP DOSYASI (ANA SITE):
cpanel-clean-upload-latest.zip

KURAL:
- Production canli yayin sadece cPanel `public_html` uzerinden yapilir.
- Tunnel URL'leri (`localhost.run`, `ngrok`, `localtunnel`, `bore`) gecici test icindir, canli kabul edilmez.
- `http://localhost:3000/` tek kaynak kabul edilir; canli yayin bunun birebir guncel hali olmalidir.

DOMAIN:
www.34motokuryeistanbul.com

CPANEL FILE MANAGER:
1) cpanel-clean-upload-latest.zip dosyasini /home/<cpanel_kullanici>/ dizinine yukle.
2) cPanel Terminal ac.
3) Asagidaki komutu tek seferde calistir.

cd ~ && TS="$(date +%Y%m%d-%H%M%S)" && mkdir -p "$HOME/backup-public_html-$TS" && if [ -d "$HOME/public_html" ]; then cp -a "$HOME/public_html/." "$HOME/backup-public_html-$TS/"; fi && rm -rf "$HOME/public_html"/* && unzip -o "$HOME/cpanel-clean-upload-latest.zip" -d "$HOME/public_html" && ls -la "$HOME/public_html" | head

SONRASI KONTROL:
- index.html kokte olmali: /home/<cpanel_kullanici>/public_html/index.html
- _next klasoru kokte olmali: /home/<cpanel_kullanici>/public_html/_next
- robots.txt ve sitemap.xml kokte olmali.

NOT:
- Bu komut once public_html yedegi alir, sonra yeni paketi acar.
- Eger unzip komutu yoksa, File Manager uzerinden zip'i Extract ederek /public_html icine ac.
